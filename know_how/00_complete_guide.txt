# JSON 데이터 처리 및 번역 작업 종합 가이드

이 문서는 JSON 형식의 데이터를 처리하고 번역하는 작업에 대한 종합 가이드입니다. 에이전트가 긴 문서를 한 번에 처리하는 데 어려움을 겪는 문제를 해결하기 위해, 가이드를 여러 개의 작은 파일로 분리했습니다.

각 파일은 특정 주제를 다루고 있으며, 파일명 앞의 숫자는 권장되는 탐색 순서입니다.

## 가이드 문서 목록

*   `01_introduction.txt`: 가이드의 목적과 전체적인 소개
*   `02_json_structure.txt`: 작업 대상인 JSON 데이터의 구조 설명
*   `03_agent_tips.txt`: AI 에이전트의 도구 사용 시 주의점 및 팁
*   `04_nodejs_scripts_basic.txt`: Node.js 스크립트를 활용한 데이터 수정의 기본 사항
*   `05_nodejs_scripts_examples.txt`: 실제 작업 사례를 바탕으로 한 스크립트 활용 예시
*   `06_nodejs_scripts_advanced.txt`: 번역 품질 향상을 위한 추가적인 팁과 고급 스크립트 활용 아이디어
*   `07_conclusion.txt`: 가이드의 결론 및 요약

필요한 정보가 어느 파일에 있는지 이 문서를 통해 빠르게 파악하고 해당 파일을 참조하시기 바랍니다.
